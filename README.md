<p align="center">Aplicação em React e Typescript para explorar repositórios do github.</p>

## Installation


```bash
$ git clone https://github.com/daviseares/github-explorer-react.git
$ cd github-explorer-react
$ yarn
```

## Run

```bash
$ yarn start
```
